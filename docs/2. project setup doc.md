# 🚀 Project Setup Guide (Next.js + Supabase + Tailwind)

## 📌 Step 0: Prerequisites

Make sure you have installed:

* Node.js (v18 or higher)
* npm or yarn
* VS Code / Antigravity

Check versions:

```
node -v
npm -v
```

---

## 📌 Step 1: Create Next.js Project

Run this command:

```
npx create-next-app@latest agency-website
```

### It will ask questions:

Select:

```
✔ TypeScript → Yes
✔ ESLint → Yes
✔ Tailwind CSS → Yes
✔ src/ directory → Yes
✔ App Router → Yes
✔ Import alias → Yes
```

---

## 📌 Step 2: Move into Project

```
cd agency-website
```

---

## 📌 Step 3: Run Project

```
npm run dev
```

Open in browser:

```
http://localhost:3000
```

✅ Now your Next.js app is running

---

## 📌 Step 4: Clean Default Files

Go to:

```
/src/app/page.tsx
```

Replace everything with:

```
export default function Home() {
  return (
    <main>
      <h1>Agency Website</h1>
    </main>
  );
}
```

---

## 📌 Step 5: Install Supabase

```
npm install @supabase/supabase-js
```

---

## 📌 Step 6: Setup Supabase Project

1. Go to Supabase website
2. Create new project
3. Copy:

   * Project URL
   * Anon Public Key

---

## 📌 Step 7: Create Supabase Client

Create file:

```
/src/lib/supabaseClient.ts
```

Add:

```
import { createClient } from '@supabase/supabase-js';

const supabaseUrl = process.env.NEXT_PUBLIC_SUPABASE_URL!;
const supabaseKey = process.env.NEXT_PUBLIC_SUPABASE_ANON_KEY!;

export const supabase = createClient(supabaseUrl, supabaseKey);
```

---

## 📌 Step 8: Add Environment Variables

Create file:

```
.env.local
```

Add:

```
NEXT_PUBLIC_SUPABASE_URL=your_url_here
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_key_here
```

⚠️ Restart server after this

```
npm run dev
```

---

## 📌 Step 9: Create Basic Pages

Inside `/src/app/` create folders:

```
services

blog
contact
```

Inside each folder create:

```
page.tsx
```

---

## 📌 Step 10: Setup Layout (Navbar + Footer)

Edit:

```
/src/app/layout.tsx
```

Basic structure:

```
export default function RootLayout({ children }) {
  return (
    <html>
      <body>
        <nav>Navbar</nav>
        {children}
        <footer>Footer</footer>
      </body>
    </html>
  );
}
```
